API Project: URL Shortener Microservice

Requirements

  • I can POST a URL to [project_url]/api/shorturl/new and I will receive a shortened URL in the JSON response. Example: {"original_url":"www.example.com","short_url": "ex01"}
  • If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}

    HINT: to be sure that the submitted url points to a valid site you can use the function dns.lookup(host, cb) from the dns core module.

  • When I visit the shortened URL, it will redirect me to my original link.

Short URL Creation

Example Usage:

[base_url]/api/shorturl/AA010101AA

Will redirect to:

http://www.google.co.jp
